# ESP8266_APwithDisplay
Sketch for ESP8266 D1 module with display SSD1306Spi
Connection:   
GND (Black) - GND.   
VCC (Red) - 3.3V.   
D0 (Orange/Brown) - D13/SCK/D5.   
D1 (Green) - D1/MOSI/D7.   
Res (Grey) - DX<-D0.   
DC (White) - D14/SDA/D1.   
CS (Grey) - GND.
